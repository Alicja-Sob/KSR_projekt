# Projekt KSR
Projekt realizowany w ramach kursu 'Komponentowe Systemy Rozproszone' w semestrze letnim 2025/2026 na Politechnice Gdańskiej.

Głównym celem projektu jest zaprojektowanie i implementacja aplikacji internetowej w której komunikacja między serwerem WWW i aplikacją backendową odbywa się z wykorzystaniem kolejek. Implementowana aplikacja będzie realizowana jako 'system dla biblioteki'

## Stack technologiczny
* .Net core/C#
